# Security Audit Report

[Xone](https://xone.org/) Chain has always regarded network security as the cornerstone of project development. We have hired industry-leading security audit companies to conduct comprehensive security audits on network architecture, smart contracts and key modules to ensure the security of user assets and ecology.

## Goal

- Ensure that the [Xone](https://xone.org/) Chain mainnet architecture is secure and reliable.

- Identify and fix potential security vulnerabilities in smart contracts and key components.

- Enhance user confidence in the [Xone](https://xone.org/) Chain network.

## Scope

The security audit covers the following areas:

1. Blockchain underlying architecture

  - Consensus

  - Network Protocol

  - Storage & State Synchronization

2. Smart Contract Security

  - Token Contract (ERC-20, ERC-721, ERC-1155 Standards)

  - Governance Related Contracts

  - DeFi Related Core Contracts (DEX, Staking)

3. Ecological Infrastructure

  - Cross-chain Bridge

  - Wallet Application

  - On-chain Governance Module

## Security Plan

- [Xone](https://xone.org/) Chain is committed to continuously investing in security resources:

- Audit and test will be conducted before each major update;

- Maintain long-term cooperation with professional audit teams;

- Regularly hold community security <Link className='text-primaryHue' isExternal showAnchorIcon href="/study/bug">vulnerability bounty programs</Link> to encourage the community to jointly maintain network security.

**We believe that security is the primary principle of public chain development.**

[Xone](https://xone.org/) Chain will continue to maintain network security and work with the community to protect every step of the growth of the ecosystem.

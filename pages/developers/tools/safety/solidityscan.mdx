# SolidityScan Overview

<Video src="video/SolidityScan.mp4"/>

<br/>

**<Link className='text-primaryHue' isExternal showAnchorIcon href="https://solidityscan.com/">SolidityScan</Link>** is a cloud-based smart contract security scanner designed specifically for **<Link className='text-primaryHue' isExternal showAnchorIcon href="https://www.soliditylang.org/">Solidity</Link>**.
It automatically detects potential vulnerabilities and anti-patterns in contracts and generates comprehensive, shareable security reports.
With its full integration into **<Link className='text-primaryHue' isExternal showAnchorIcon href="https://xone.org/">Xone Chain</Link>**, developers can now use <Link className='text-primaryHue' isExternal showAnchorIcon href="https://solidityscan.com/">SolidityScan</Link> directly within the Xone ecosystem to enhance pre-deployment and post-deployment contract security.

## Key Features

* **Automated Contract Auditing** — Supports scanning both deployed contracts and source code to identify potential security risks.
* **Security Scoring** — Generates an overall “Security Score” for each contract, helping developers assess security posture at a glance.
* **Multi-Chain Compatibility** — Works seamlessly with major EVM-compatible blockchains, including **<Link className='text-primaryHue' isExternal showAnchorIcon href="https://xone.org/">Xone Chain</Link>**.
* **API & SDK Support** — Provides REST, WebSocket, CLI, and SDK integrations to fit into modern CI/CD pipelines.
* **Detailed Reports & Recommendations** — Includes vulnerability types, severity levels, code locations, and actionable remediation guidance.

## Integration with Xone Chain

### Platform & Chain Parameters

To use <Link className='text-primaryHue' isExternal showAnchorIcon href="https://solidityscan.com/">SolidityScan</Link> on **<Link className='text-primaryHue' isExternal showAnchorIcon href="https://xone.org/">Xone Chain</Link>**, specific platform and chain parameters must be set in each API request. <Link className='text-primaryHue' isExternal showAnchorIcon href="https://apidoc.solidityscan.com/solidityscan-security-api/getting-started/platform-parameters/xone/blockscout">More >></Link>

* `contract_platform`: `blockscout`
* `contract_platform_id`: `16`
* For **Xone Mainnet**:

  * `contract_chain`: `xone-mainnet`
  * `contract_chain_id`: `82`
* For **Xone Testnet**:

  * `contract_chain`: `xone-testnet`
  * `contract_chain_id`: `83`

### Example HTTP Request

```bash showLineNumbers copy
curl --location --request GET 'https://api.solidityscan.com/api/v1/<scan_type>/<contract_platform_id>/<contract_chain_id>/0x4ee38aa8d7449a177119e983610D73e9ace932dA' \
--header 'Authorization: Token {{token}}'
```

### Example Websocket Message

```json showLineNumbers copy
{
    "action": "message",
    "payload": {
        "type": "<ws_scan_type>",
        "body": {
            "contract_address": "0xc2498D4fB17A956A847091D088f1D7f6fE745f5B",
            "contract_platform": "<contract_platform>",
            "contract_chain": "<contract_chain>"
        }
    }
}
```

### Recommended Integration Flow

1. Obtain or link your **API Token** on the <Link className='text-primaryHue' isExternal showAnchorIcon href="https://solidityscan.com/">SolidityScan</Link> platform.
2. Configure platform and chain parameters (as shown above), specifying the deployed contract address on **<Link className='text-primaryHue' isExternal showAnchorIcon href="https://xone.org/">Xone Chain</Link>**.
3. Execute the scan to receive the contract’s security score and full vulnerability report.
4. Integrate the scanning process into your **CI/CD pipeline** for continuous security monitoring.

## Advantages

* **Efficiency** — Automated scanning drastically reduces audit turnaround time and supports both source-level and on-chain analysis.
* **Accuracy** — Combines static analysis with AI-based heuristics for precise vulnerability detection across multiple networks.
* **Transparency** — Each report details every finding, risk category, and recommended fix for better developer insight.
* **Ecosystem Integration** — Fully compatible with **<Link className='text-primaryHue' isExternal showAnchorIcon href="https://xone.org/">Xone Chain</Link>**, allowing native use without migration or configuration overhead.

## Use Cases

* Pre-deployment security audits for contracts on **<Link className='text-primaryHue' isExternal showAnchorIcon href="https://xone.org/">Xone Chain</Link>**.
* Periodic re-scans of deployed contracts for continuous security assurance.
* Integration into DApp/DeFi development pipelines for automated safety checks.

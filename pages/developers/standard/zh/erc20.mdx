# 使用 Remix 和 OpenZeppelin 部署 ERC-20 代币

本指南将引导您使用 **Remix**（一个在线 Solidity IDE）和 **OpenZeppelin**（一个包含安全且可重用的智能合约模板的库）来部署一个 ERC-20 代币。

## 先决条件

1.  打开 <Link className='text-primaryHue' isExternal showAnchorIcon href="链接1">Remix IDE</Link>。
2.  安装 <Link className='text-primaryHue' isExternal showAnchorIcon href="链接2">MetaMask</Link> 扩展程序，并将其配置为连接到 Xone 测试网。
3.  从 <Link className='text-primaryHue' isExternal showAnchorIcon href="链接3">水龙头</Link> 获取测试网 XOC 代币，为您的 MetaMask 钱包提供资金。

## 开始操作

<Steps>
### 步骤 1：创建 ERC-20 合约

1.  在 Remix 中，于 `contracts` 文件夹内创建一个新文件，并将其命名为 `example.sol`。

2.  将以下代码添加到 `example.sol` 文件中：

``` js showLineNumbers copy filename="example.sol"
// SPDX-License-Identifier: MIT
// Compatible with OpenZeppelin Contracts ^5.0.0
pragma solidity ^0.8.22;

import "@openzeppelin/contracts/token/ERC20/ERC20.sol";
import "@openzeppelin/contracts/token/ERC20/extensions/ERC20Permit.sol";

contract Test is ERC20, ERC20Permit {
    constructor() ERC20("example", "EXAMPLE") ERC20Permit("example") {
        _mint(msg.sender, 10000 * 10 ** decimals());
    }
}

```
<ImageViewer src="/images/Create ERC-20.png" alt="Create ERC-20" />

<Callout type="info" emoji="ℹ️">
   修改构造函数中的参数以自定义代币：

   **name:** 您的代币名称。`example`

   **symbol:** 您的代币符号。`EXAMPLE`

   **initialSupply:** 您的代币初始供应量 10000（以整个单位计，在合约内部会乘以 10^decimals()）。

   在此示例中，代币名称为 "example"，符号为 "EXAMPLE"，初始供应量为 10,000 个代币。
</Callout>

### 步骤 2：编译合约

1.  点击左侧边栏中的【**Solidity Compiler**】标签页。
2.  选择适当的编译器版本（0.8.x）。
3.  点击【**Advanced Configuration**】以展开高级设置。
4.  选择 EVM 版本为【**paris**】。
5.  启用优化（默认值为 200）。
6.  点击【**Compile example.sol**】。

<ImageViewer src="/images/Create ERC-20_1.png" alt="Create ERC-20_1" />

### 步骤 3：部署合约

1.  点击左侧边栏中的【**Deploy & Run Transactions**】标签页。
2.  如果您使用的是 MetaMask，请将环境设置为 **Injected Provider - MetaMask**；如果使用模拟环境，则设置为 **JavaScript VM**。
3.  点击【**Deploy**】。

<ImageViewer src="/images/Create ERC-20_2.png" alt="Create ERC-20_2" />

### 步骤 4：展开已部署的合约

1.  在 **Deployed Contracts** 部分，找到您刚刚部署的合约。
2.  点击合约名称将其展开。这将显示您可以与之交互的可用函数和变量列表。
3.  检查智能合约地址（例如：<Link className='text-primaryHue' isExternal showAnchorIcon href="链接4">0xb1a04485443d937e8eEbd260AE954BC3C006422F</Link>）。

<ImageViewer src="/images/Create ERC-20_3.png" alt="Create ERC-20_3" />

### 步骤 5：将代币添加到 MetaMask

1.  打开 MetaMask 并点击【**Import Tokens**】。
2.  粘贴您部署的代币的合约地址（本例中为 `0xb1a04485443d937e8eEbd260AE954BC3C006422F`）。
3.  MetaMask 将自动检测代币符号和小数位数。点击【**Add Custom Token**】。
4.  检查您的代币余额。

<ImageViewer src="/images/Create ERC-20_4.png" alt="Create ERC-20_4" />

### 步骤 6：恭喜！您已完成 ERC-20 合约的创建。

</Steps>

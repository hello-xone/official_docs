# RPC Endpoints

This document provides the necessary details to connect to the Xone Testnet, including network configuration, RPC endpoints, and block explorer information.

## **Testnet Configuration**

### **Network Details**
- **Network Name:** Xone TestNet  
- **New RPC URL:** [https://rpc.xonetest.plus/](https://rpc.xonetest.plus/)  
- **Chain ID:** 33772211  
- **Symbol:** TXOC  
- **Block Explorer URL:** [https://testnet.xscscan.com/](https://testnet.xscscan.com/)  

### **How to Connect to Xone Testnet**

#### **For MetaMask**
1. Open MetaMask and click on your profile icon.
2. Navigate to **Settings** > **Networks** > **Add Network**.
3. Enter the following details:

![add testnet to metamask](public/images/metamask_testrpc.png)

4. Save the network and start interacting with the Xone TestNet.

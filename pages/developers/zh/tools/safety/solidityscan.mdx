# SolidityScan 概述

<Video src="video/SolidityScan.mp4"/>

<br/>

**<Link className='text-primaryHue' isExternal showAnchorIcon href="链接1">SolidityScan</Link>** 是一款专为 **<Link className='text-primaryHue' isExternal showAnchorIcon href="链接2">Solidity</Link>** 设计的云端智能合约安全扫描器。
它能自动检测合约中的潜在漏洞和反模式，并生成全面、可共享的安全报告。
凭借其与 **<Link className='text-primaryHue' isExternal showAnchorIcon href="链接3">Xone Chain</Link>** 的完全集成，开发者现在可以直接在 Xone 生态系统内使用 <Link className='text-primaryHue' isExternal showAnchorIcon href="链接1">SolidityScan</Link>，以增强部署前和部署后的合约安全性。

## 主要特性

*   **自动化合约审计** — 支持扫描已部署合约和源代码，以识别潜在安全风险。
*   **安全评分** — 为每个合约生成总体"安全评分"，帮助开发者一目了然地评估安全状况。
*   **多链兼容性** — 可与主要的 EVM 兼容区块链无缝协作，包括 **<Link className='text-primaryHue' isExternal showAnchorIcon href="链接3">Xone Chain</Link>**。
*   **API 和 SDK 支持** — 提供 REST、WebSocket、CLI 和 SDK 集成，以适应现代 CI/CD 流水线。
*   **详细报告和建议** — 包含漏洞类型、严重级别、代码位置和可操作的修复指导。

## 与 Xone Chain 的集成

### 平台和链参数

要在 **<Link className='text-primaryHue' isExternal showAnchorIcon href="链接3">Xone Chain</Link>** 上使用 <Link className='text-primaryHue' isExternal showAnchorIcon href="链接1">SolidityScan</Link>，必须在每个 API 请求中设置特定的平台和链参数。<Link className='text-primaryHue' isExternal showAnchorIcon href="链接4">了解更多 >></Link>

*   `contract_platform`: `blockscout`
*   `contract_platform_id`: `16`
*   对于 **Xone 主网**:
    *   `contract_chain`: `xone-mainnet`
    *   `contract_chain_id`: `82`
*   对于 **Xone 测试网**:
    *   `contract_chain`: `xone-testnet`
    *   `contract_chain_id`: `83`

### HTTP 请求示例

```bash showLineNumbers copy
curl --location --request GET 'https://api.solidityscan.com/api/v1/<scan_type>/<contract_platform_id>/<contract_chain_id>/0x4ee38aa8d7449a177119e983610D73e9ace932dA' \
--header 'Authorization: Token {{token}}'
```

### WebSocket 消息示例

```json showLineNumbers copy
{
    "action": "message",
    "payload": {

        "body": {

            "contract_platform": "<contract_platform>",
            "contract_chain": "<contract_chain>"
        }
    }
}
```

### 推荐的集成流程

1.  在 <Link className='text-primaryHue' isExternal showAnchorIcon href="链接1">SolidityScan</Link> 平台上获取或关联您的 **API 令牌**。
2.  配置平台和链参数（如上所示），指定在 **<Link className='text-primaryHue' isExternal showAnchorIcon href="链接3">Xone Chain</Link>** 上部署的合约地址。
3.  执行扫描以接收合约的安全评分和完整的漏洞报告。
4.  将扫描过程集成到您的 **CI/CD 流水线** 中，以实现持续的安全监控。

## 优势

*   **高效性** — 自动化扫描大大缩短了审计周期，并支持源代码级和链上分析。
*   **准确性** — 结合静态分析和基于 AI 的启发式方法，跨多个网络精确检测漏洞。
*   **透明度** — 每份报告都详细说明了每个发现、风险类别和推荐的修复方法，以便开发者更好地理解。
*   **生态系统集成** — 与 **<Link className='text-primaryHue' isExternal showAnchorIcon href="链接3">Xone Chain</Link>** 完全兼容，无需迁移或配置开销即可原生使用。

## 使用场景

*   在 **<Link className='text-primaryHue' isExternal showAnchorIcon href="链接3">Xone Chain</Link>** 上部署合约前的安全审计。
*   对已部署合约进行定期重新扫描，以实现持续的安全保障。
*   集成到 DApp/DeFi 开发流水线中，实现自动化安全检查。
*   在 **Xone 生态系统** 内进行项目提交或上币验证，以增强信任度和可靠性。

## Web3

Returns general network configuration data for Xone nodes.

[web3_clientVersion](/openapi/web3/web3_clientVersion) : Returns the version of the Xone node client in use. \
[web3_sha3](/openapi/web3/web3_sha3) : Computes the Keccak-256 (SHA3) hash of the provided data.

These methods are essential for node management and cryptographic operations on Xone.

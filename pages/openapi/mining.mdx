## Mining

Provides details on the mining state of the [Xone](https://xone.org/) node.

[eth_coinbase](/openapi/mining/eth_coinbase) : Returns the address of the miner who is receiving mining rewards. \
[eth_mining](/openapi/mining/eth_mining) : Indicates whether the node is actively mining blocks.

These methods are useful for monitoring the mining status of an [Xone](https://xone.org/) node.

## Transactions info

Retrieves data on individual transactions, such as status, receipt, and inclusion in a specific block.

[eth_getTransactionByHash](/openapi/transactionsinfo/eth_getTransactionByHash) : Fetches transaction details using its hash.  \
[eth_getTransactionCount](/openapi/transactionsinfo/eth_getTransactionCount) : Returns the number of transactions sent from an address. \
[eth_getTransactionReceipt](/openapi/transactionsinfo/eth_getTransactionReceipt) : Retrieves the receipt of a processed transaction.  \
[eth_newPendingTransactionFilter](/openapi/transactionsinfo/eth_newPendingTransactionFilter) : Available only on paid tier. Sets up a filter for monitoring pending transactions. \
[eth_getTransactionByBlockHashAndIndex](/openapi/transactionsinfo/eth_getTransactionByBlockHashAndIndex) : Fetches a transaction by block hash and index. \
[eth_getTransactionByBlockNumberAndIndex](/openapi/transactionsinfo/eth_getTransactionByBlockNumberAndIndex) : Retrieves a transaction by block number and index. \
[txpool_content](/openapi/transactionsinfo/txpool_content) : Provides the contents of the transaction pool.

These methods assist in transaction tracking and management on [Xone](https://xone.org/).
